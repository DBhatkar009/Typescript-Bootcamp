Importing the fs module: This module provides an API for interacting with the file system.
readTextFile function: This function takes a file path as an argument and reads the file asynchronously.
fs.readFile method: This method reads the file content. The first argument is the file path, the second is the encoding (in this case, 'utf8'), and the third is a callback function that handles the file content or error.
Error handling: If there’s an error reading the file, it logs the error to the console.
Logging the file content: If the file is read successfully, it logs the content to the console.
Make sure to replace './example.txt' with the path to your actual text file.
